    Tentang

    Aplikasi kecil ini dibuat untuk menunjukkan praktik UI modern: responsif, interaksi dinamis, dan persistensi data lokal. Cocok sebagai starter atau portofolio.

    • Interaktif: Tambah, edit, hapus, dan filter tugas secara langsung.
    • Persistensi: Data tersimpan di browser (localStorage) sehingga tetap ada setelah reload.
    • Desain: Responsif, dengan micro-interaksi, loader, dan animasi halus.
